Resolves #1. Implements zero-copy conversion between `ethnum::U256` and `Fr`
field elements across the workspace:

- `ZkError` enum with `InvalidFieldElement` variant.
- `Fr` newtype wrapping `ethnum::u256` with private invariant.
- `SafeFrom<u256>` trait using constant-time `overflowing_sub` range check — no heap allocation, no panics.
- `HostConvert` trait on `Env`: `fr_from_u256(&self, U256) -> Result<Fr, ZkError>`, all paths `#[inline(always)]` to minimise WASM stack depth.
- 6 unit tests (zk-core) + 5 integration tests (zk-soroban) covering zero, boundary (r-1), modulus, and `u256::MAX`.
